# Domain

Knowledge about the world the code models: MUDs in general, the wire protocols nogfx speaks, and Achaea / Iron Realms specifics.

- [`mud.md`](mud.md) — general MUD vocabulary (rooms, prompt, realms, players).
- [`telnet.md`](telnet.md) — the telnet protocol, focused on what nogfx actually implements.
- [`gmcp.md`](gmcp.md) — the Generic MUD Communication Protocol, including the telnet-level wrapping and the messages nogfx handles.
- [`achaea/`](achaea/INDEX.md) — Achaea-specific knowledge: cities, classes, skills, combat, channels.
